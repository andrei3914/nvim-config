### Installation

Clone this into the ~/.config/nvim directory:
```bash
git clone git@github.com:andrei3914/nvim-config.git ~/.config/nvim
```

Navigate to the directory and open it with nvim:
```bash
cd
cd ~/.config/nvim
nvim .
```

You're good to go!
